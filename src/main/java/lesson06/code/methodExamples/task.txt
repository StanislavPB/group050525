Написать программу, которая отмечает день рождения кошки.
Требования:
1) создайте отдельный класс для кошки
2) Создайте две переменных : имя и возраст
3) Передайте первоначальные данных в эти переменные
4) создайте два метода
    - который печатает информацию о кошке
    - который "отмечает ее день рождения"

В методе main:
- создать новую кошку передав ее данные
- вывести информацию о ней
- отметь ее день рождения
- еще раз вывести информацию о кошке
